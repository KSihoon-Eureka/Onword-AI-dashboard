# 11st Future Channel API Note

## Status

Official OpenAPI information exists for 11st order-related services, but detailed endpoint use is gated by seller/OpenAPI registration.

## Known details

- Official entry point: `https://openapi.11st.co.kr/openapi/OpenApiServiceIntroduce.tmall?introduceType=ORDER`
- Access model: seller OpenAPI key/registration required.
- Expected integration shape: polling order list/detail APIs by date/status after credentials are available.

## Connector implications

- Do not add to MVP UDS mapping until a field dictionary is extracted from credentialed docs/API samples.
- Keep portal Excel fallback available for early operations.
- Ask seller/11st 담당자 for: order list endpoint, order detail endpoint, claim/return endpoints, rate limits, date window, pagination, and settlement API availability.
